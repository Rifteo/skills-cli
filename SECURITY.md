# Security Policy

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Report vulnerabilities privately using [GitHub's Security Advisories](../../security/advisories/new).

We will acknowledge within 48 hours and aim to resolve within 14 days.

## Scope

**In scope:**
- Vulnerabilities in the CLI code
- Issues that could allow malicious content to be written to agent config files
- Dependency vulnerabilities with active exploits

**Out of scope:**
- Vulnerabilities in systems that installed skills describe testing
- Issues with third-party tools referenced in skill steps
- Security issues in your own environment or agent setup
